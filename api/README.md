# Python Networking & API Fundamentals

This project explores Python scripting, REST APIs, microservices, and data formats such as CSV and JSON. It focuses on consuming APIs, handling structured data, and writing clean, maintainable Python code that follows PEP 8 and Pythonic naming conventions.

## Learning Objectives

By completing this project, you will understand:

* The limitations and appropriate use cases of Bash scripting
* What APIs are and how they work
* REST API architecture and principles
* The concept of microservices
* CSV and JSON data formats
* Pythonic naming conventions for:

  * Packages and modules
  * Classes
  * Variables
  * Functions
  * Constants
* The significance of CapWords (CamelCase) in Python development

## Project Requirements

* Developed and tested on Ubuntu 14.04 LTS
* Uses Python 3.4.3
* Follows PEP 8 style guidelines
* Executable Python scripts with proper shebang (`#!/usr/bin/python3`)
* Alphabetically organized imports
* Comprehensive module documentation
* Safe dictionary access using `get()`
* Proper use of `if __name__ == "__main__":` to prevent code execution on import
* Includes a mandatory README.md at the project root

This repository demonstrates best practices for Python scripting while introducing core concepts in modern web services and API consumption.
